# Description
This Crash-Class, part of the Ettore Majorana Workshop “Complexity in Mental Health: The Interplay Between Brain, Behavior, and Context” (Erice, 26 Apr–1 May 2026), offers hands-on training in network analysis with real-world data, enabling participants to explore and interpret complex networks in mental health research.
